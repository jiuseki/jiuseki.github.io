自从秋招签了荔支后就开摆了，浑浑噩噩的享受最后的自由时光大半年，其实一直隐隐处在焦虑之中，是那种自己无所事事但感觉若继续下去将会遭到报应的焦虑。

\n

最近一次受到暴击，是看到其他人的考研结果。初中同学考到了北邮计算机、考到了中科院微电子所，都是很牛逼，非常牛逼的。他们都是平时很低调的人，以至于发出朋友圈时让人非常意外。但我心里越来越难受了。我挣扎了很久是否要考研，直至现在我也还不确定直接工作的选择是否正确，他们的结果让我的犹疑更甚，我的选择真的正确吗？因为他们过去都不是成绩过分优秀的人。甚至可以说，我很多时候可能比他们成绩更好，而我在学习上是那么的要强，我多想考研到好学校的是我。

\n

可是我也知道，我考研的决心可能比不上他们十分之一，我学习的刻苦习惯在大学四年间已经蒸发，在漫长的迷茫阶段我已经失去了方向，这样的我即使考研可能也不会取得好的结果。所以一切可能一切都是在更早的时候注定了。

\n

可能是在我选择专业的时候，我选择了网新这个前路茫然的新专业，以至于不知道考研应该学什么，不知道工作应该做什么；可能是在我对暨南大学的摆烂接纳中，我不是那么满意却又不觉得应追求更好，所以我没有刻舟求剑、往更高处去的念头；可能是在这个自由轻松的环境中，我过了太久舒适的生活，而没有刻苦的同龄人在push着我。当然也可能是更多我不知道的事情，可能只是我以为我与某些人在同一条起跑线上，而实则他们早已有其他天赋与资本。

\n

我记得很久之前，我就在知乎上看到过这样的话，大概是有的人用表面努力来迷惑自己，那时候我还在疑惑，怎么有人做这么奇怪的事情。但是，回顾我的前四年，看似忙碌却不知道在忙什么，总是用没时间考虑、现在有ddl来推脱，我对我自己的人生压根缺乏规划和思考，我总是在最后一刻的靠悔恨和焦虑作为动力来奋力一跳，触底微弹后又继续缓慢下滑。

\n

温水煮青蛙。我真的努力了吗？我真的用心了吗？我不见得能于心无愧地回答。在人生的选择上，我总是走向更轻松的那个。
